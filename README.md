
# Tech Stack
	•	Vue 3 + Vite
	•	Leaflet
	•	leaflet.markercluster
	•	Tailwind CSS
	•	Node.js Backend

# Data Sources
	•	Static Nodes: Fetched from /static-nodes (usually enriched admin_peers)
	•	Live Nodes: Fetched from /live-nodes (from Linea EthStats WS)
	•	Uses AWS region heuristics + IP geolocation for mapping
